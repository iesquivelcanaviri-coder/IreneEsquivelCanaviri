
# Laugh Lines: The Evolution of Comedy**

## Overview

**Laugh Lines** is a fully responsive, multi-page website created for the **Web Design module (12% assessment)**.
The site explores **the evolution of comedy**, from ancient theatre to modern social-media humour, using semantic HTML5, responsive CSS, modern layout techniques, and interactive UI components.

This project demonstrates:

* Semantic, accessible HTML
* CSS Grid + Flexbox layouts
* Responsive design using %, vw, rem
* Embedded video + media integration
* Horizontal scrolling components
* Structured multi-page navigation
* Consistent visual identity across all pages

## **Design & Technologies Used**

### **HTML5**

* Semantic layout (`header`, `nav`, `main`, `article`, `section`, `aside`, `footer`)
* Accessible structure for screen readers

### **CSS3**

* Flexbox (layouts, side-by-side content, cards)
* CSS Grid (home page layout)
* Sticky sidebar
* Horizontal scroll containers
* Responsive images and typography
* Fluid units: %, vw, vh, rem
* Media queries for mobile, tablet, desktop

### **Accessibility (WCAG-aligned)**

* Strong colour contrast (black background, white text)
* Alt text for all images
* Logical heading hierarchy
* Keyboard-friendly navigation
* Clear, descriptive link text
