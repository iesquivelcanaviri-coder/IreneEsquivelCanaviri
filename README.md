# Laugh Lines: The Evolution of Comedy
_Web Design Assignment — Module 1_

---

## Project Overview

**Laugh Lines** is a responsive, multi-page educational website developed for the **Web Design module (12% assessment weighting)**.  
The project explores the evolution of comedy from ancient theatrical traditions to modern digital humour across global cultures and social media platforms.

The website was designed, developed, validated, and deployed using **HTML5 and CSS3**, following recommended web design practices and fully addressing the **assessment criteria and lecturer feedback**.

# Alignment with Assessment Criteria

---

## 1. Creating a Hosted Website with HTML & CSS

**Assessment requirement:**  
Demonstrate an understanding and apply key concepts and principles of creating a hosted website with HTML and CSS.

**How this project meets the requirement:**

- All pages use **semantic HTML5**
- A single **external CSS file** controls layout, typography, and colour
- No inline or embedded CSS is used
- The website is successfully **hosted on GitHub Pages**
- Homepage correctly named **index.html**



---

## 2. Modern Page Layout Design

**Assessment requirement:**  
Show clear understanding of how to design a modern page layout with HTML and CSS.

**Implemented features:**

- Fixed and responsive navigation bar
- Flexbox layouts for:
  - Two-column sections
  - Sidebar + main content
  - Card-based layouts
- Scroll-snap horizontal carousel
- Sticky sidebar navigation
- Consistent header, navigation, and footer across all pages



---

## 3. Tables and Structured Data

**Assessment requirement:**  
Show clear understanding of key concepts of creating tables with HTML and CSS.

**Implementation:**

- One page uses **HTML tables** for structured information
- Tables include `<thead>`, `<tbody>`, `<th>`, and `<td>`
- Tables styled using CSS only
- Responsive table containers prevent layout issues on smaller screens



---

## 4. Responsive Web Design

**Assessment requirement:**  
Demonstrate ability to create responsive web pages that display well in multiple form factors.

**Responsive techniques used:**

- Media queries at **1024px and 768px**
- Fluid typography using `rem`, `%`, and `clamp()`
- Responsive images with `max-width: 100%`
- Horizontal scroll for wide content
- Mobile-friendly navigation (hamburger menu)
- Responsive YouTube embeds

**Tested at:**

- 375px (mobile)
- 768px (tablet)
- 1024px (small laptop)
- 1366px+ (desktop)
